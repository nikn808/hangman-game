# Виселица 🎮

Консольная игры "Виселица". Игрок должен угадать загаданное слово. За каждую неверную попытку рисуется элемент виселицы.

## Правила игры

- Компьютер загадывает случайное слово
- Игрок пытается угадать слово, вводя буквы или слова целиком
- При неверной букве теряется попытка и рисуется виселица
- При неверном слове поражение
- Игра продолжается до угадывания слова или исчерпания 6 попыток
- После 3 ошибок, открывается текстовая подсказка

## Формат файла words.json

**Структура объекта:**
- `word` *(строка)* — загаданное слово. Состоит только из русских букв.
- `hint` *(строка)* — небольшая текстовая подсказка, помогающая игроку.
